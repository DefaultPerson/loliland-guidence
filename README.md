# TechnoMagic RPG — интерактивный чеклист прохождения

Статический веб-гайд по прохождению модпака **LoliLand «TechnoMagic RPG»** (Minecraft 1.7.10) для прохождения вдвоём. Прогресс хранится локально в браузере (`localStorage`), сборки нет — открывается как `file://` или раздаётся любым статик-сервером.

**Живая версия:** https://tm-894cb569.hydramonitors.com/

## Что внутри

- **Карта** — обзор прохождения: радиальный прогресс, волны 1–5 сверху вниз, карточки этапов с игровыми значками.
- **Все этапы** — сайдбар + детальный экран этапа с задачами, чипами предметов (с игровыми текстурами, блоки — изометрией как в инвентаре), заметками Сервер / Кооп / Оптимизация / Привилегия.
- 15 разделов / ~165 шагов, выверенных по плейлисту прохождения и извлечённым ресурсам клиента.

## Структура

| Путь | Назначение |
|------|------------|
| `index.html`, `app.js`, `styles.css`, `fonts.css` | приложение |
| `guide.js` | контент гайда (генерится `meta/build_guide.py` из `data/workflow_result.json`) |
| `item_icons.js`, `data/item_icons.json` | имя предмета → текстура |
| `icons/` | извлечённые игровые текстуры (значки этапов, чипы, iso-кубы) |
| `fonts/` | локальный Exo 2 (woff2, кириллица) |
| `data/` | исходные данные: имена, idmap, рецепты, lang, вики |
| `meta/*.py`, `meta/*.js` | пайплайн сборки контента и иконок |

## Пайплайн иконок (требует ресурсов игры в `game/`)

```
meta/extract_all_textures.py   # все текстуры+lang из jar модпака → meta/tex, data/lang
meta/build_item_icons.py       # имя → текстура (lang-мост + IDF-фаззи + оверрайды)
meta/normalize_icons.py        # обрезка анимированных лент до первого кадра
meta/render_iso_blocks.py      # блоки → изометрия (inventory-style)
```

Ресурсы игры (`game/`, ~1 ГБ) в репозиторий не коммитятся (`.gitignore`).

> Точное дерево квестов хранится на сервере — сверяйся с квест-буком BetterQuesting в игре.
